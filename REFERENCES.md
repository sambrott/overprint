# External references (Overprint tools)

Used for UX and implementation ideas — Overprint stays **vanilla JS**, no copied code.

## Beat maker / sequencer
- **MDN — Advanced techniques: Creating and sequencing audio**  
  https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Advanced_techniques  
  Scheduling, envelopes, patterns.
- **ordrumbox-v2** (vanilla Web Audio drum machine)  
  https://cadeli.github.io/ordrumbox-v2/
- **beatbox.js** (TypeScript sequencer library — API ideas only)  
  https://github.com/beatboxjs/beatbox.js

## Color / contrast
- **WCAG 2 contrast** (formulas)  
  https://www.w3.org/WAI/WCAG21/Understanding/contrast-minimum.html

## QR codes
- **node-qrcode** (algorithm / options reference)  
  https://github.com/soldair/node-qrcode

## Image / canvas
- MDN **Canvas API** — resizing, `toBlob`, `createImageBitmap`
